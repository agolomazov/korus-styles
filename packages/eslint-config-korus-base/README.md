# eslint-config-korus-base

This package provides Korus's .eslintrc as an extensible shared config.

## Usage

### Install package:
```shell script
npm install --save-dev eslint-config-korus-base 
```
### Extend your config .eslintrc
```json
{
  "extends": [
    "eslint-config-korus-base"
  ]
}
```
